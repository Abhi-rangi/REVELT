# REVELT

**AI-powered systems that capture, convert, and book more patients automatically.**

REVELT is a business automation platform. It handles every inquiry, follow-up, and booking so your team can focus on care — with instant 24/7 responses, automated appointment booking, and fewer missed calls and no-shows.

## What’s in this repo

Marketing and product site for REVELT: hero, social proof, problem/solution, stats, about, how it works, services, FAQ, demo section, and footer. Built with Next.js, React, Tailwind CSS, and Radix UI (shadcn-style components).

---

### Run the project

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

**Production:**

```bash
npm run build
npm start
```
